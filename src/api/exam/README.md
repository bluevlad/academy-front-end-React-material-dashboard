# Exam API

시험(Exam) 관리와 관련된 API 함수들을 정의합니다.

## 📝 주요 기능

- **시험 관리**: 시험 목록 조회, 상세 조회
- **시험 응시**: 모의고사 응시 및 결과 확인

## 🛠 함수 목록

- `fetchExamList(params)`: 시험 목록을 조회합니다.
- `fetchExamDetail(params)`: 시험 상세 정보(문제 등)를 조회합니다.
- `submitExam(data)`: 시험 답안을 제출합니다.
- `fetchExamResult(params)`: 시험 결과를 조회합니다.
